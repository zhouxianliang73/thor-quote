# THOR索而橱柜报价网页 — v1 开发任务

## 目标
创建一个纯静态报价网页（单页HTML），复刻 `报价模板-2026-00.xlsx` 的报价结构，数据基于 `THOR索而家居厨柜类价格表-2025.xlsx` 和 `五金.xlsx`。

## 产出位置
`CAB\01-System\Codex\thor-quote-v1\index.html`
（已经建好目录）

## 功能需求

### 1. 页面布局
- 中国风/现代简约风格，A4纸打印友好
- 顶部：公司LOGO区域（文字"THOR 索而智能家居"）+ 报价表标题
- 客户信息区：报价日期、下单店面、订单号、客户姓名、客户地址、联系电话、客户电话
- 主体：按品类分8个区域
- 底部：合计金额、折扣、签字区、条款

### 2. 品类区域

#### ① 柜体（核心！）
- 选择系列：下拉选（大千云纹、大悦韵纹、大影臻纹、大彩绚丽、大观金属、大语玉润、大传质影、铂金、铂金光学膜、云纹、韵纹、臻纹、绚丽、金属、玉润、质影/原色）
- 选柜型：下拉选（洗手盘地柜/地柜/灶台吊柜/吊柜/高柜/上翻门吊柜）
- 选具体规格：下拉选（地柜680*570 / 地柜720*570 / 吊柜700*370 / 吊柜800*370 / 吊柜900*370）
- 填数量（米）
- 填尺寸（宽*高*深）
- 非标系数：默认1.0，可改
- 门板系列：自动根据选的系列填充
- 单价：根据系列+规格自动从价格表匹配
- 金额 = 单价 × 数量 × 非标系数
- 可添加多行（点击"添加行"按钮）

### 3. 合计与折扣
- 自动汇总所有品类小计
- 折扣选择：门店价(8.5折) / 活动价(8折) / 店长价(7.5折) / 老板价(7折)
- 显示折后金额
- 人民币大写自动生成

### 4. 打印/导出
- 使用 `@media print` CSS 适配A4纸打印
- 页面底部有"打印报价单"按钮
- 打印时隐藏按钮、隐藏多余的编辑控件

### 5. 样式要求
- 专业报价单风格，白底黑字，干净整洁
- 表格样式：带边框，表头灰色背景
- 字体：宋体/微软雅黑
- A4纸一页以内展示（内容多时可分页）
- 底部包含：总公司信息（索而智能家居有限公司，东莞市洪梅镇海新路2号）、条款说明

## 技术要求
- 纯 HTML + CSS + JavaScript，单文件
- 无需外部依赖
- 所有数据硬编码在JS中
- 兼容 Chrome 打印

## 验收标准
1. 打开页面就能用，无需任何服务端
2. 选系列→选柜型→自动出单价
3. 填数量→自动算金额
4. 所有品类都有，合计正确
5. 折扣计算准确
6. 打印出来像正式报价单
