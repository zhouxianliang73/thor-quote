# THOR 选品页 · Phase 1 开发任务

> 参考：D:\CAB-sites\supplier-selector
> 目标：每个板块独立选品页，容器化展示产品

---

## 核心功能清单

### 1. 选品页结构
```
┌────────────────────────────────────────┐
│ 板块导航栏（柜体/台面/配件/水槽/电器…）  │
├────────────────────────────────────────┤
│ 筛选栏                                  │
│  分类/材质/颜色/尺寸 筛选标签            │
├────────────────────────────────────────┤
│ 搜索栏                                  │
│  输入关键词搜索产品                      │
├────────────────────────────────────────┤
│ 产品网格                                  │
│  ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐     │
│  │ 图  │ │ 图  │ │ 图  │ │ 图  │     │
│  │编号  │ │编号  │ │编号  │ │编号  │     │
│  │名称  │ │名称  │ │名称  │ │名称  │     │
│  │尺寸  │ │尺寸  │ │尺寸  │ │尺寸  │     │
│  └─────┘ └─────┘ └─────┘ └─────┘     │
├────────────────────────────────────────┤
│ 选品确认 → 加入报价单 → 返回编辑器       │
└────────────────────────────────────────┘
```

### 2. 产品详情卡片
```
点击产品卡片 → 详情浮层浮在页面最上方
  ├── 大图（可左右切换多张）
  ├── 编号 / 名称
  ├── 规格尺寸 / 材质 / 颜色
  ├── 技术参数表格
  ├── 使用场景图（轮播）
  └── "加入报价单" 按钮
```

### 3. 数据源
- 从 THOR_DATA 读取产品数据（现有）
- 扩展：增加产品详情、使用场景、多角度图字段
- 图片路径：`03-Converted/01-images/{category}/`

### 4. 与报价单联动
- 选品确认后，数据填入当前报价单的对应板块
- 支持多选、改数量
- 返回编辑器后自动刷新
